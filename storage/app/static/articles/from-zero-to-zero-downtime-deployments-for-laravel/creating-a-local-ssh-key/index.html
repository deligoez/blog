Creating a local SSH Key

2 min 205 words

8 months ago January 6th, 2021

Go to your .ssh directory or create the directory if it is not exists yet.

cd ~/.ssh

Create an SSH key pair

ssh-keygen -o -a 100 -t ed25519 -f id_ye_deligoz_me -C "ye@deligoz.me"

Enter a strong password

Reenter your password

This will generate two files. One for public and one for the private key

id_ye_deligoz_me
id_ye_deligoz_me.pub
